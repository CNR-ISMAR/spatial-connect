about=SpatialConnect is a demonstrative QGIS plugin for spatial connectivity propagation.
    It applies a transition matrix T (e.g. from a Lagrangian particle-tracking model)
    to an input raster x to compute the distribution after n time steps:

        x(t+n·dt) = x(t) · T^n

    Supported matrix formats: MatrixMarket (.mtx) and NumPy sparse (.npz).

    The plugin handles masked domains (land/sea mask) via cell-ID mapping so
    that the matrix size N does not need to equal rows × cols of the raster.

    Typical use cases: ocean pollution dispersal, fishing effort footprint,
    larval connectivity, habitat fragmentation.
